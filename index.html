❓ AIDE RAPIDE
📌 Pin / Épingler
Clique sur le bouton 📌 à côté d'un build, loot, note ou carte World State pour l'afficher sur ton Dashboard. Clique à nouveau pour retirer.
⏱ Timers
Les timers (Sortie, Baro, Arbitration, Nightwave) affichent le temps restant. Rouge = moins de 1 h, Orange = moins de 6 h.
🏷 Tags
Ajoute des tags (ex: Steel Path, Eidolon) aux builds et au loot pour mieux les organiser. Filtre ensuite par tag dans les champs dédiés.
🔍 Filtres
Utilise les champs de recherche (nom, notes) et le filtre tags pour affiner la liste des builds et du loot.
🌓 Thème
Le bouton 🌓 en haut à droite bascule entre mode sombre (par défaut) et mode clair.
⬆⬇ Export / Import
Sauvegarde tes données en JSON ou restaure-les depuis un autre appareil. Données stockées localement dans ton navigateur.
